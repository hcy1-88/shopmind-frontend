# ShopMind - AI 智能电商平台（前端）

ShopMind 是一个融合大模型能力的智能电商系统，除了最基本的购物功能外，还聚焦于在**搜索页、商品详情页、订单页**提供上下文感知的 AI 助手，实现自然语言导购、商品问答与订单咨询。

## 技术栈

- Vue 3（组合式 API + `<script setup>`）
- Vite
- TypeScript
- Pinia（状态管理）
- Vue Router
- Element Plus（UI 组件库）
- Axios（HTTP 请求）

## 启动和构建命令

```bash
npm install

npm run dev

npm run build
```
